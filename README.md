# 💼 Developer Portfolio

A modern, responsive developer portfolio built with React to showcase my software engineering projects, technical skills, and professional journey.

The portfolio serves as a central hub for my work, highlighting featured projects, a comprehensive project archive, technical expertise, and contact information in a clean, user-friendly interface.

---

## ✨ Features

- 📱 Fully responsive design
- ⚛️ Built with React and Vite
- 🧩 Component-based architecture
- 📂 Featured projects with GitHub and live demo links
- 📚 Project archive showcasing my learning journey
- 🛠️ Technical skills section
- 📖 Professional experience and learning journey
- 📄 Downloadable CV
- 📬 Contact section with direct links to GitHub, LinkedIn, email, and phone
- 🎨 Modern UI using CSS Modules

## 💻 Running Locally

Clone the repository:

```bash
git clone https://github.com/adenya-arn/portfolio.git
```

Navigate into the project:

```bash
cd portfolio
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```
